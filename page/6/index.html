Python可变与不可变数据类型

发表于 2019-06-04 本文字数： 2.5k 标签： Python Valine：

首先必须理解的是，python万物皆对象，其中一切的传递都是对象引用（地址）的传递，无论是赋值还是函数调用，不存在值传递。

python变量是无类型的，使用无需事先声明但必须赋值，我们所说的”类型”是变量引用的对象的类型，这个引用指向堆内存里的对象，在堆中分配的对象分为两类：

  • 不可变对象：Number（数值）、String（字符串）、Tuple（元组），变量值改变、变量地址必须改变

  • 可变对象：Set（集合）、List（列表）、Dictionary（字典)，变量值改变、变量地址不改变

不可变对象的内容不可改变，保证了数据的不可修改（安全，防止出错），同时可以使得在多线程读取的时候不需要加锁。

Http与https协议相关笔记

发表于 2019-05-06 本文字数： 2.7k 标签： 随笔 Valine：

HTTP，HyperText Transfer Protocol，超文本传输协议，通常运行在TCP上、工作于应用层（OSI的最上层）、默认80端口，是浏览器与服务器之间以明文的方式传送内容的一种互联网通信协议，HTTP协议是基于B/S架构通信的。

HTTPS，Hyper Text Transfer Protocol over SecureSocket Layer，是在HTTP的基础上主要基于SPDF协议结合SSL/TLS加密协议，客户端依靠证书验证服务器身份传递加密信息的通信协议，默认443端口。

HTTP通信模型

HTTP协议采用了请求/响应模型

  • 由HTTP客户端发起一个请求，创建一个到服务器指定端口（默认是80端口）的TCP连接。
  • HTTP服务器则在那个端口监听客户端的请求。一旦收到请求，服务器会向客户端返回
    • 一个状态，比如”HTTP/1.1 200 OK”，
    • 以及返回的内容，如请求的文件、错误消息、或者其它信息。

Python内置函数·字典方法

发表于 2019-05-03 本文字数： 987 标签： Python Valine：

Python中字典（dict）是使用{}包裹的键值对集合，是一种无序的可变类型（元素改变字典地址不改变）。

  • 键（key）必须为不可变类型，如str、int。
  • 值可为任意对象（甚至可以foo）。
  • 典型字典：d = {'name': 'tom', 'age':15, 'store':[84, 65, 91]}

快速生成字典

使用 dict.fromkeys(iterable[, value=None])方法可以快速生成以iterable对象中元素为键，值为value的字典

d1 = dict.fromkeys(range(5))
print(d1)   # {0: None, 1: None, 2: None, 3: None, 4: None}

d2 = dict.fromkeys(range(5), 'yes')
print(d2)   # {0: 'yes', 1: 'yes', 2: 'yes', 3: 'yes', 4: 'yes'}
