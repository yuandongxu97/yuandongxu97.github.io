Python设计模式：单例模式

发表于 2019-08-04 本文字数： 5.8k 标签： Python 设计模式 Valine：

单例模式是面向对象的编程语言中23种设计模式之一，属于创建型设计模式：

  • 目的：一个类在一个进程中只存在一个实例对象。
  • 用途：减少对象的频繁创建与销毁的消耗（如数据库连接）、某些实例的硬性要求（如打印机）。
  • 代码特点：构造器私有化以保证只能由类自行创建对象、需要提供全局访问点获取单例对象。
  • 缺点：单例对象为全局变量需要注意被修改、容易高耦合等

Python中单例模式的实现方法有很多种：

  • import模块导入
  • 装饰器类
  • 单例类内类方法
  • 基于类内置的__new__方法实现（推荐方法）
  • 基于元类（metaclass）实现

单例模式原理详见百度百科： 单例模式（Java）

通常我们说的Python高阶函数指的是函数的参数类型为函数，或者函数的返回值类型为函数。

Python中常用的高阶函数有：

  • map(func, *iterables)：序列元素处理
  • filter(function or None, iterable)：序列元素处理与过滤
  • reduce(function, sequence[, initial])：序列阶乘化处理
  • partial(func, *args, **keywords)：为函数参数指定默认值

Vi/vim的使用笔记

发表于 2019-06-11 本文字数： 1.5k 标签： Linux Valine：

vi是unix中默认的文本编辑器，虽然看起来简单但功能相当强大，vim可以看作是vi的pro升级版本，添加了代码高亮、系统兼容等多种特性，更适合coding。

Vim通常分为三种模式：

命令模式：Vim默认进入命令模式，此时所有的键盘敲击都会被识别为命令而不是字符输入，而编辑器则处于等待用户输入命令的状态。

输入模式：即插入模式，通常使用此模式来编辑文本，当按下Esc键时自动退出输入模式，并进入命令模式。

底线命令模式：在命令模式中输入冒号:就可以进入底线命令模式了，输入命令后回车即可执行对应的命令并退出底线命令模式，当按下Esc键时也会自动退出底线命令模式，并进入命令模式。
