Django(二)·Hello World

发表于 2020-05-05 本文字数： 780 标签： Django

Django 采用了 MVT 的软件设计模式，即模型（Model），视图（View）和模板（Template）。

1. 创建第一个网页

1.1 新建页面

打开boards应用程序中的views.py文件，并添加以下代码：

from django.http import HttpResponse

def index(request):
    s = "<h2>hello, world!</h2>"
    return HttpResponse(s)

视图是接收httprequest对象并返回一个httpresponse对象的Python函数。接收 request 作为参数并返回 response 作为结果。

1.2 定义路由

我们必须告诉Django什么时候会调用这个view。这需要在项目的urls.py文件中完成：

from django.conf.urls import url
from django.contrib import admin

from boards import views

urlpatterns = [
    url(r'^$', views.index, name='index'),
    url(r'^admin/', admin.site.urls),
]

Django使用正则表达式来匹配请求的URL。对于我们的home视图，我使用^$ 正则，它将匹配一个空路径，也就是主页（这个URL：http://127.0.0.1:8000 ）。

如果我想匹配的URL是 http://127.0.0.1:8000/indexpage/ ，那么我的URL正则表达式就会是：url(r'^indexpage/$', views.index, name='index')。

1.3 测试

python manage.py runserver

Web浏览器中打开 http://127.0.0.1:8000
