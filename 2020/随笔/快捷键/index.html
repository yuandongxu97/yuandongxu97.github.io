小技巧

在终端窗口命令提示符下，连续按两次 Tab 键、或者连续按三次 Esc 键、或者按 Ctrl+I 组合键，将显示所有的命令及工具名称。

Ubuntu 系统切换到字符界面： \Ctrl+Alt+Fn**（n=1…6）；切换回图形界面： Ctrl+Alt+F7 或 Alt+F7。

CentOS 系统切换到字符界面： Ctrl+Alt+Fn（n=2…6）；切换回图形界面： Ctrl+Alt+F1 或 Alt+F1。
