Hexo博客使用经验

发表于 2020-01-04 本文字数： 2.5k 标签： 随笔

Hexo使用过程中的一些经验

博客搭建

博客生成

Hexo GitHub项目地址：https://github.com/hexojs/hexo

安装Hexo

$ npm install hexo-cli -g

创建博客

$ hexo init blog
$ cd blog

博客使用

$ hexo clean         # 清理缓存（public文件夹）
$ hexo generate      # 生成博客静态文件
$ hexo server        # 开启调试本地服务器

主题安装

安装Next 5：

$ cd hexo-blog
$ git clone https://github.com/iissnan/hexo-theme-next themes/next

安装Next 6&7：

$ cd hexo-blog
$ git clone https://github.com/theme-next/hexo-theme-next themes/next

博客优化

1. 常规配置

1.1 prism代码高亮

hexo自带highlight、primsjs代码高亮插件

2. 网页样式修改

next主题配置文件blog/themes/next/ _config.yml，取消行注释

custom_file_path:
  ···
  style: source/_data/styles.styl

新建文件夹及文件：blog/source/_data/styles.styl

2.1 添加网页背景图片

body {
    background:url(/assets/background.jpg);        // 背景图片位置
    background-repeat: no-repeat;        // 若果背景图片不能全屏，那么是否平铺显示，充满屏幕
    background-attachment: fixed;        // 背景是否随着网页上下滚动而滚动，fixed为固定
    background-position:50% 50%;        // 位置
    opacity: 0.9;
    background-size:cover;
}

2.2 博客网页宽度自适应

调整line1、3的百分数值可以调整网页宽度

header{ width: 80% !important; }
header.post-header {
  width: auto !important;
}
.container .main-inner { width: 80%; }
.content-wrap { width: calc(100% - 260px); }

.header {
  +tablet() {
    width: auto !important;
  }
  +mobile() {
    width: auto !important;
  }
}

.container .main-inner {
  +tablet() {
    width: auto !important;
  }
  +mobile() {
    width: auto !important;
  }
}

.content-wrap {
  +tablet() {
    width: 100% !important;
  }
  +mobile() {
    width: 100% !important;
  }
}

3. 扩展功能（extension）

3.1 本地搜索

安装包

$ cd blog
$ npm install hexo-generator-searchdb

Hexo站点配置文件

search:
  path: search.xml
  field: post
  format: html
  limit: 10000

Next主题配置文件

# Local Search
# Dependencies: https://github.com/theme-next/hexo-generator-searchdb
local_search:
  enable: true
