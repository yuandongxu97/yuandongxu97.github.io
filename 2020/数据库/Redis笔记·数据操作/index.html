Redis数据库·数据结构与使用命令

发表于 2020-01-11 本文字数： 1.1k 标签： 数据库

redis是键值对数据库，它的键类型只能为字符串（string），但其值的类型却有5种：字符串string、哈希hash、列表list、集合set、有序集合zset。

键操作

keys *            # 查询键列表，*为pattern
exists key        # 存在否
type key        # 查询类型
del key1 key2 ...    # 删除键
expire key seconds    # 设置过期时间
ttl key                # 查看有效时间

String字符串

字符串类型是Redis中最为基础的数据存储类型，它在Redis中是二进制安全的，这便意味着该类型可以接受任何格式的数据，如JPEG图像数据或Json对象描述信息等。在Redis中字符串类型的Value最多可以容纳的数据长度是512M。

set key value
mset key1 value1 key2 value2 ...    # 设置多个键值

setex key seconds value        # 设置键值及过期时间，以秒为单位
append key value            # 追加

get key
mget key1 key2 ...            # 获取多个

# 值长度
strlen key

HashMAP

可以将hash类型的key视为一个对象，其value就是对象的属性及其值的键值对。

hkeys key                        # 获取键的所有属性列表
hvals key                        # 获取所有属性值
hget key field                    # 获取属性值
hmget key field1 field2 ...        # 获取多个属性值
hdel key field1 field2 ...        # 删除属性
hmset key field1 value1 field2 value2 ...        # 设置值

List列表

列表的元素类型为string，按照插⼊顺序排序

lpush [···, ···, ···, ]    rpush        # 添加
lpop  [···, ···, ···, ]    rpop        # 删除
lpush key v1 v2 v3
lpop key
# 返回列表⾥指定范围内的元素
lrange key start stop    

# 在指定元素的前或后插⼊新元素
linsert key before或after 现有元素 新元素    

# 设置指定索引位置的元素值
lset key index value

# 将列表中前count次出现的值为value的元素移除
# count > 0: -->
# count < 0: <--
# count = 0: 移除所有
lrem key count value

Set集合

⽆序、元素为string类型、元素具有唯⼀性，不重复、对于集合没有修改操作

# add
sadd key member1 member2 ...

# 返回all
smembers key

# 删除指定
srem key

Zset有序集合

  • sorted set，有序集合
  • 元素为string类型
  • 元素具有唯⼀性，不重复
  • 每个元素都会关联⼀个double类型的score，表示权重，通过权重将元素从⼩到⼤排序
