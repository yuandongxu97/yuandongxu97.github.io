Docker部署httpbin网络测试服务

发表于 2019-01-03 本文字数： 3.1k 标签： Docker

httpbin是大神kennethreitz为搭建测试用的http服务而写的一个开源项目，httpbin 这个服务应用能测试HTTP请求及响应的各种信息，比如 cookie、ip、headers 和登录验证等，且支持 GET、POST 等多种方法，对 web 开发和测试具有很大的帮助。

Docker部署

假设你服务器的公网IP为1.2.3.4，那么部署成功后httpbin服务应该开放在1.2.3.4:81上

docker pull kennethreitz/httpbin
docker run -d -p 81:80 --restart=always kennethreitz/httpbin

Python使用案例

比如说测试一个代理的有效性，可以使用

import requests

# 待测试代理
proxy = 'http://10.11.12.13:8888'

log = None

try:
    log = requests.get('http://aliyun:81/ip', proxies={'http': proxy}, timeout=5).text
except:
    log = "无效代理"

print(log)      # 无效代理
