Python内置函数·列表方法

发表于 2019-05-01 本文字数： 2.1k 标签： Python Valine：

Python中列表（list）是使用中括号包裹的元素集合，是一种可变类型（元素改变地址不改变），使用列表的内置方法可以快速对列表进行处理。

列表推导

l = [1, 2, 3]
foo = [x for x in l.__dir__() if '_' not in x]
print(foo)
# ['clear', 'copy', 'append', 'insert', 'extend', 'pop', 'remove', 'index', 'count', 'reverse', 'sort']

其中：

  • l为典型列表，foo为使用推导式生成的列表数据类型内置的方法列表
  • l.__dir__()：输出对象内容，包括属性、方法、实现等
  • if '_' not in x：不输出其中包含下划线的元素（即类似 __str__）

列表方法

  • l.clear()：清除列表内元素，使列表为空。

    Out[52]: [1, 2, 3]
l.clear()
l
Out[54]: []

  • l1 = l.copy()：复制列表到一个新变量。

    l = [1, 2, 3]
l1 = l.copy()
l1
Out[51]: [1, 2, 3]

  • l.append(obj)：将元素添加至列表尾部。

    l.append(l1)
l
Out[16]: [1, 2, 3, [1, 2, 3]]

  • l.insert(index, obj)：将元素添加到索引位置。

    l.insert(1, l1)
l
Out[18]: [1, [1, 2, 3], 2, 3, [1, 2, 3]]

  • l.extend(iterable)：将可迭代对象中的元素依次添加到列表尾部。

    l = [1, 2, 3]
l1 = l.copy()
l.extend(l1)
l
Out[32]: [1, 2, 3, 1, 2, 3]

  • l.pop()：从列表为弹出一个元素，这个元素是返回值，可赋值给新对象。

    l = [1, 2, 3]
l.pop()
Out[40]: 3
a = l.pop()
a
Out[42]: 2

  • l.remove(value)：从列表头开始查找元素，删除一次。

    l = [1, 2, 3, 1, 2]
l.remove(2)
l
Out[59]: [1, 3, 1, 2]

  • index：列表头开始查找，返回第一个匹配的索引。

    l = [1, 2, 3, 1, 2]
l.index(2)
Out[61]: 1

  • count：返回元素个数。

    l = [1, 2, 3, 1, 2]
l.count(2)
Out[63]: 2

  • reverse：反转列表，即逆序。

    l = [1, 2, 3, 1, 2]
l.reverse()
l
Out[66]: [2, 1, 3, 2, 1]

l.sort(<list>, key=None)

这个方法用于列表内元素的就地排队，如例：

l = [4, 1, 3, 5, 2, 3]
print(id(l))
l.sort()    # 排序
print(l)
print(id(l))
# 2607261377096
# [1, 2, 3, 3, 4, 5]
# 2607261377096

可以看到，地址并未改变。

并且这个方法强大之处在于还可以接受一个排序依据参数：key

l = [
    'c, 20190604, 91',
    'b, 20190506, 60',
    'a, 20190705, 77'
]

l.sort(key=lambda x: x.split(',')[0])
print(l)    # ['a, 20190705, 77', 'b, 20190506, 60', 'c, 20190604, 91']
l.sort(key=lambda x: x.split(',')[1])
print(l)    # ['b, 20190506, 60', 'c, 20190604, 91', 'a, 20190705, 77']
l.sort(key=lambda x: x.split(',')[2])
print(l)    # ['b, 20190506, 60', 'a, 20190705, 77', 'c, 20190604, 91']

其中 lambda是Python的匿名函数，旨在快速的编写一次性方法，具有简短精悍的特点

l.sort(key=lambda x: x.split(',')[0])
# 作用同
def sort_by(x):
    return x.split(',')[0]
l.sort(key=sort_by)

作用同

l = [
    'c, 20190604, 91',
    'b, 20190506, 60',
    'a, 20190705, 77'
]

def sort_by(x):
    return x.split(',')[0]

l.sort(key=sort_by)

print(l)    # ['a, 20190705, 77', 'b, 20190506, 60', 'c, 20190604, 91']
