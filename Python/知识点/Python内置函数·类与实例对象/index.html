Python内置函数·类与实例对象

发表于 2019-05-11 本文字数： 244 标签： Python Valine：

使用实例如下

class A:
     pass

a = A()

查看函数文档方法

obj.__class__：创建对象（type())

因为python万物皆对象，相对于类的实例对象是由类创建的来说，类对象也应该有一个创建者，那就是type（类的创建者）

A.__class__
Out[4]: type
a.__class__
Out[5]: __main__.A

obj.__doc__：文档

obj.__dict__：内容字典

obj.__module__：所属模块（文件）

obj.__dir__：对象的属性、方法、实现列表
